## What technologies are used for this project?

This project is built with:
- Vite
- TypeScript
- React

# Features:
Contact form with options to send via email, WhatsApp, or Instagram
Mission, vision, and values ​​sections
Responsive design for mobile and desktop
Light/dark mode
Social media: Instagram, Facebook, and TikTok
Purpose: To convert visitors into clients by showcasing event planning services with a focus on creating "unique and memorable experiences."
